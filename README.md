## HR Analytics Dashboard

This repository contains the Tableau HR Analytics Dashboard that provides insights into key workforce metrics such as employee demographics, performance, and attrition trends.
The workbook file (.twbx) is included for reference and can be opened using Tableau Desktop / Tableau Public Desktop.

# Interactive Dashboard

To view the fully interactive version of the dashboard, visit the Tableau Public link below:
